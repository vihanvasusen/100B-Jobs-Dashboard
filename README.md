# 100B Jobs - AI Hiring Dashboard

An AI-powered candidate selection dashboard built with **Flask** and **Bootstrap**, featuring:
- Upload candidate JSON
- Filter by Role, Location, and Skills
- Score candidates
- AI-recommended top 5 candidates with reasoning in Markdown
